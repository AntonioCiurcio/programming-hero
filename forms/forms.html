<FORM>

Formularios são a melhor foma de conseguirmos informações dos usuarios e assim estabelecermos uma relação mais cordial e amistosa.
A tag usada par formularios é a <form> e seus principais atributos são:

    action
    Define o local onde os dados coletados serão alocados, geralmente é a url do server
    method
    Normalmente seguido pelo velor get ou poste define o metodo de envio ao server.

Para aceitarmos um dado do usuario usamos a tag <input/> e devemos sempre colocar atributos que tipificam e indentificam os dados, o principais atributo é:

    type
    indica o tipo de conteudo a ser preenchido

E os principais tipos de type são:
    password
    usado para esconder caracteres pois normalmente é onde senhas sao colocadas
    submit
    cria um botão de envio
    e-mail
    cria a aceitação de e-mail's que passam por parametros especificos
    tel
    cria uma area que aceita telefone
    checkbox
    cria um botão check
    radio
    cria botoes de seleção
